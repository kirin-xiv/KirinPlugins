# Kirin's FFXIV Plugin Collection

A collection of original Dalamud plugins for Final Fantasy XIV, focused on party games, social automation, and roleplay tools.

**Repository URL:** https://raw.githubusercontent.com/kirin-xiv/KirinPlugins/main/pluginmaster.json

---

## Installation

1. Launch FFXIV with Dalamud enabled
2. Open the Dalamud Settings (type `/xlsettings` in chat)
3. Navigate to the **Experimental** tab
4. Add the following custom repository URL:

```https://raw.githubusercontent.com/kirin-xiv/KirinPlugins/main/pluginmaster.json```

5. Save and close settings
6. Open the Plugin Installer (`/xlplugins`)
7. Search for the desired plugin and click **Install**

## Plugins

### Truth or Dare (FFToD)
**Version:** 2.3.0.1

Fully automated Truth or Dare game management for party entertainment.

#### Features
- One-click game initialization with automatic rule posting
- Multi-winner support (1-3 players)
- Customizable announcement messages
- Theme system for personalized game experiences
- Winner-specific chat channel support
- Smart tiebreaker system
- Duplicate winner prevention

---

### Spamroll Giveaway
**Version:** 2.2.0.0

Automated giveaway management with intelligent winner detection.

#### Features
- Configurable winning number parameters
- Multi-winner support for larger giveaways
- Anti-spam protection mechanisms
- Multiple chat channel integration
- Automatic winner announcement

---

### EorzeanScribe
**Version:** 1.6.5.0 (Beta)

Writing assistant designed for roleplayers and content creators.

#### Features
- Text segmentation for long-form content
- Built-in spell checking
- Integrated thesaurus
- Direct-to-chat posting capability
- 16,000 character editor workspace
- Writing statistics and analytics
- Customizable user interface

> **Note:** This plugin is currently in beta. Please report any issues you encounter.
